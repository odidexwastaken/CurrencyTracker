# CurrencyTracker - Döviz Takip Uygulaması

Bu proje, Final Ödevi kapsamında **Frankfurter API** kullanılarak geliştirilmiş bir C# Konsol uygulamasıdır.

## Özellikler
- **API Entegrasyonu:** `https://api.frankfurter.app` üzerinden anlık veriler çekilir.
- **LINQ Sorguları:** Veriler üzerinde filtreleme, sıralama ve istatistik işlemleri yapılır.

## Nasıl Çalıştırılır?
Projeyi indirin ve Visual Studio ile açarak çalıştırın.
